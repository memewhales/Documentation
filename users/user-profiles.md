# User Profiles

## Overview

Every user on Yoink has a personal profile that showcases their trading activity, holdings, and platform engagement. User profiles provide transparency and enable the community to track performance and achievements.

## Accessing User Profiles

### Your Own Profile

Access your profile by:

1. Click your wallet address in the header
2. Navigate to "My Profile" from menu
3. Or visit: `yoink.gg/profile/[your-address]`

### Other User Profiles

View other users by:

* Clicking on usernames in chat
* Selecting from leaderboards
* Clicking on trader names in activity feeds
* Direct URL if you have their wallet address

## Profile Settings

### Customization Options

#### Personal Information
* **Display Name**: Set a custom username
* **Bio**: Write a short introduction
* **Avatar**: Upload profile picture (or use NFT)
* **Social Links**: Connect Twitter, Discord, etc.

#### Privacy Settings
* **Public Profile**: Show or hide from search
