# Streamer Profiles

Discover creator profiles to learn about streamers, view their trading history, and explore their created tokens. Each verified streamer has a comprehensive profile showcasing their activity and achievements on Yoink.

## Profile Overview

When you click on a streamer's profile, you'll see a detailed dashboard with multiple sections:

### Header Information

**Profile Picture & Username**
- Large circular profile image from connected social accounts
- Display name and username (e.g., "KimarieSky" with @KimarieSky handle)
- Profile verification status indicators

**Verification Badge**
{% hint style="info" %}
The verification badge (‚úì) is only visible if the streamer has:
- Completed the verification process
- Successfully linked their wallet to their profile
- Met all verification requirements
{% endhint %}

**Action Buttons**
- **"Claim & Verify Profile"** - For unverified streamers to start verification
- **"Follow/Subscribe"** - Connect with the streamer's content

### Key Statistics

**Subscriber Metrics**
- **Subscribers** - Number of people following this streamer
- **Subscribing** - Number of accounts this streamer follows
- Shows community engagement and network size

**Trading Performance**
- **Total Volume** - Cumulative trading volume across all transactions
- **Creator Revenue** - Total earnings from created tokens and trading fees
- **Forked Coins** - Number of tokens created by this streamer
- **Joined Date** - When the streamer first joined Yoink (shows "N/A" for new profiles)

### Profile Sections

**About Section**
- Bio information (shows "No bio available" if not filled)
- Personal description and background
- Links to external social media accounts

**Recent Trades**
- Live feed of the streamer's latest trading activity
- Shows "No trades found" for new or inactive accounts
- Displays transaction history when available

**Earnings Overview**
- Detailed breakdown of revenue sources
- Expandable section with comprehensive earning metrics
- Charts and graphs showing performance over time

## Verification Status

{% hint style="warning" %}
**üö® Important: Profile Verification Process**

**Anyone can submit** a profile verification for any streamer, but **only the Yoink team** can approve verifications.

**Submitted vs. Verified Profiles:**
- **Submitted Profile** ‚Üí Someone created a profile for a streamer, but this doesn't mean the real creator is on the platform. We're just representing them based on public information.
- **Verified Profile** ‚Üí The Yoink team has manually checked and received proof that the streamer is actually on the platform and the profile belongs to them.

**What verification means:**
- ‚úÖ Real streamer confirmed their presence on Yoink
- ‚úÖ Identity verified by Yoink team
- ‚úÖ Profile ownership authenticated
- ‚úÖ Safe to interact with and support

Always look for the verification badge before assuming a streamer is actively using Yoink!

### Verified Streamers
- Display verification checkmark badge
- Have completed identity verification
- Wallet successfully linked to profile
- Enhanced profile features available
- Increased trust and credibility

### Unverified Streamers
- No verification badge shown
