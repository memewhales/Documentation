## Step-by-Step Guide

### 1. Find a Token

Navigate to the token you want to buy:

* **Browse All Coins**: View complete token list
* **Search**: Use search bar for specific creators
* **Leaderboard**: See top performing tokens
* **Live Streams**: Find currently streaming creators

### 2. Review Token Details

Before buying, check:

* **Chart**: Price trend and volume
* **Market Cap**: Current valuation
* **Holders**: Number of token holders
* **Early Bird Seats**: Available seats (X/20)
* **Creator**: Verified status and info
* **Trading Activity**: Recent buys/sells

{% hint style="success" %}
**Early Bird Opportunity**: If seats are available (< 20/20), you can become an Early Bird by being among the first 20 buyers!
{% endhint %}

### 3. Open Buy Widget

On the token page:

1. Locate the **Buy/Sell widget** (usually on right side)
2. Ensure **"Buy"** tab is selected
3. Widget shows your current SOL balance

### 4. Enter Amount

Choose how much to spend:

#### Option A: Enter SOL Amount

* Type the amount of SOL you want to spend
* Example: `0.5` SOL
* Widget calculates tokens you'll receive

#### Option B: Use Quick Amounts

* Click preset buttons: `0.1`, `0.5`, `1`, `5` SOL
* Fast way to buy common amounts

#### Option C: Buy Percentage

* Use `25%`, `50%`, `75%`, `100%` buttons
* Buys percentage of your available SOL balance

### 5. Review Trade Details

The widget displays:

| Detail | What It Means |
|--------|---------------|
| **You Pay** | SOL amount you're spending |
| **You Receive** | Tokens you'll get (estimated) |
| **Price Impact** | How your trade affects price |
| **Slippage Tolerance** | Max acceptable price change |
| **Trading Fee** | Platform and creator fees |
| **Minimum Received** | Worst-case token amount |

{% hint style="warning" %}
**Check Price Impact**: High price impact (>5%) means you're moving the market significantly. Consider splitting large orders.
{% endhint %}

### 6. Adjust Slippage (Optional)

Click slippage settings icon:

* **Low Liquidity**: Set 3-10% slippage
* **High Liquidity**: Set 0.5-2% slippage
* **Large Orders**: Increase slippage tolerance
* **Default**: Usually 1% works well

### 7. Confirm Purchase

1. Click **"Buy"** button
2. Review transaction in wallet popup
3. Check transaction details match
4. Approve/Confirm in your wallet
5. Wait for confirmation (10-30 seconds)

### 8. Transaction Confirmed

Once confirmed:

* âœ… Tokens appear in your wallet
* âœ… Balance updates in your profile
* âœ… Transaction shows in history
* âœ… Early Bird status updated (if applicable)

## Transaction Errors

### Insufficient SOL

**Error**: "Insufficient funds"

**Solution**:
* Add more SOL to wallet
* Reduce buy amount
* Account for gas fees

### Slippage Exceeded

**Error**: "Slippage tolerance exceeded"

**Solution**:
* Increase slippage tolerance
* Wait for price to stabilize
* Reduce order size
* Try again in a few seconds

### Transaction Failed

**Error**: "Transaction failed"

**Solution**:
* Check network status
* Ensure wallet is connected
* Refresh page and retry
* Contact support if persists

### Price Changed

**Error**: "Price has changed significantly"

**Solution**:
* Review new price
* Adjust amount if needed
* Confirm you're okay with new price
* Or cancel and wait
