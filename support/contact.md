## What to Include

### For All Issues

When contacting support, provide:

* ‚úÖ **Your wallet address** (first and last 4 chars: 0x1234...5678)
* ‚úÖ **What you were doing** (step by step)
* ‚úÖ **What went wrong** (exact error message)
* ‚úÖ **What you expected** to happen
* ‚úÖ **Screenshots** if applicable
* ‚úÖ **Browser and wallet** you're using

### For Transaction Issues

**Additionally include**:

* Transaction hash (if available)
* Token symbol/name
* Amount attempted
* Time of transaction
* Error message exact text

### For Account Issues

**Additionally include**:

* When account was created
* Verification status
* Twitch/social usernames
* Recent account changes
