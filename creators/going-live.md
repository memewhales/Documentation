## Troubleshooting

### LIVE Badge Not Appearing

**Checklist**:

* âœ… Actually live on Twitch?
* âœ… Twitch account connected?
* âœ… Verified creator status?
* âœ… Token exists?
* âœ… Good standing on platform?

**Solutions**:

1. Refresh token page
2. Manually trigger in Creator Dashboard
3. Disconnect and reconnect Twitch
4. Wait 1-2 minutes for sync
5. Contact support if persists

### Stream Not Embedding

**Common Issues**:

* Twitch set to "Subscriber Only"
* VODs disabled
* Geo-restrictions active
* Twitch API issues

**Fix**:

* Ensure stream is public
* Enable embedding in Twitch settings
* Check Twitch dashboard settings
* Try different browser

### Low Viewer Engagement

**Improve Engagement**:

* More interactive content
* Respond to chat actively
* Display token info prominently
* Create holder-exclusive content
* Host giveaways/events

### Trading Volume Not Increasing

**Potential Reasons**:

* Need more viewers
* Weak call-to-action
* Token already at high market cap
* Poor stream timing
* Content not compelling

**Solutions**:

* Grow Twitch audience first
* Better promote token during stream
* Create urgency (limited offers)
* Stream during peak hours
* Improve content quality
