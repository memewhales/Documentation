# How to Create a Token

Learn how to create a creator token on Yoink - for yourself or for another creator.

## Prerequisites

Before creating a token, ensure you have:

* ✅ Connected Solana wallet
* ✅ At least 0.5 SOL (for deployment and liquidity)
* ✅ High-quality creator image (recommended: 512x512px)
* ✅ Token details prepared (name, symbol, description)

## Step-by-Step Guide

### 1. Navigate to Token Creation

* Click **"Create"** in the top navigation
* Or visit the Create Token page directly

### 2. Upload Creator Image

{% hint style="info" %}
**Image Requirements**:
* Format: PNG, JPG, or GIF
* Size: 512x512px recommended
* Max file size: 5MB
* Clear, high-quality image
{% endhint %}

The image will represent the token across the platform:

* Token cards
* Trading pages
* Leaderboards
* Social shares

### 3. Enter Token Details

#### Token Name

* Full creator name or brand name
* Example: "DrDisrespect" or "The Doc"
* 3-50 characters
* Should be recognizable

#### Token Symbol

* Short 3-5 character ticker
* Example: "DOC" or "2TIME"
* Uppercase letters recommended
* Should be memorable and unique

#### Description

Write a compelling description including:

* Who the creator is
* Content style and niche
* Streaming schedule
* Why people should trade this token
* Links to social media

**Example**:
```
The Two-Time! DrDisrespect is the most dominant gaming streamer on the planet. 
Entertaining millions with high-octane FPS gameplay, unforgettable characters, 
and championship-level content. Streams daily at 2PM PST.

Follow: @DrDisrespect
```

### 4. Add Creator Information

#### Twitch Username (Optional but Recommended)

* Link the creator's Twitch account
* Enables automatic stream detection
* Shows live status on platform
* Required for "Live" features

#### Social Links (Optional)

* Twitter/X
* YouTube
* Discord
* Instagram
* TikTok

### 5. Set Initial Parameters

#### Initial Supply

* Default: 1,073,000,000 tokens
* Standard across all tokens
* Cannot be changed after creation

#### Bonding Curve Target

* When hit, token graduates to Raydium
* Creates permanent liquidity
* Default target based on platform settings

### 6. Review and Deploy

Before deploying:

* ✅ Double-check all information
* ✅ Verify image looks good
* ✅ Confirm you have enough SOL
* ✅ Understand deployment costs

{% hint style="warning" %}
**Important**: Token details CANNOT be changed after deployment! Review carefully.
{% endhint %}

### 7. Confirm Transaction

1. Click **"Create Token"**
2. Review transaction in wallet
3. Approve the transaction
4. Wait for confirmation (10-30 seconds)

### 8. Token is Live!

Once confirmed:

* Token appears on platform immediately
* Trading begins right away
* You become holder #1 (if you buy)
* Share with community!

## After Creation

1. **Buy First**: Consider being the first buyer (Early Bird #1!)

3. **Tag Creator**: Let them know about their token
4. **Monitor**: Watch trading activity

1. **Verify Account**: Link your Twitch/socials

## Best Practices

### Choosing Creators

Create tokens for creators who:

* Have active, engaged communities
* Stream regularly and consistently
* Produce quality content
* Have growth potential
* Are active on multiple platforms

### Token Naming

* Use creator's real name or widely-known alias
* Keep ticker symbols clear and memorable
* Avoid confusing similar tokens
* Check if creator already has a token

### Images

* Use official profile pictures when possible
* Ensure image is clear and recognizable
* Avoid copyrighted material you don't own
* Consider asking creator permission

### Descriptions

Write descriptions that:

* Accurately represent the creator
* Include relevant keywords for search
* Provide value to potential traders
* Link to creator's socials
* Mention streaming schedule if known

## Common Issues

### Transaction Failed

**Causes**:
* Insufficient SOL for fees
* Network congestion
* Rejected transaction

**Solutions**:
* Ensure you have 0.5+ SOL
* Wait a few minutes and retry
* Check wallet is connected
* Try refreshing page

### Image Upload Failed

**Causes**:
* File too large
* Unsupported format
* Network issues

**Solutions**:
* Compress image under 5MB
* Convert to PNG or JPG
* Check internet connection
* Try different image

### Duplicate Token

If a token already exists:

* Search before creating
* Consider supporting existing token instead
* Contact support if you believe you should be the official creator
